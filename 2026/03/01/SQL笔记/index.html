SQL笔记

SQL（结构化查询语言）的不同功能分类：数据定义语言(DDL)、数据查询语言(DQL)、数据操纵语言(DML)、数据控制语言(DCL)和事务控制语言(TCL)

类别 名称 功能 常用关键词
DDL 数据定义语言 定义结构 (表、库、索引) CREATE, ALTER, DROP, TRUNCATE
DQL 数据查询语言 数据查询 SELECT
DML 数据操纵语言 数据增、删、改 INSERT, DELETE, UPDATE
DCL 数据控制语言 权限控制 GRANT, REVOKE
TCL 事务控制语言 事务管理 COMMIT, ROLLBACK, SAVEPOINT
